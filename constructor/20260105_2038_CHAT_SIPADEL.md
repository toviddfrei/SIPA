## 📊 RESPUESTA OBTENIDA

Ese error de "No such file" ocurre porque al lanzar SIPA desde la shell, la ruta relativa ./test_db_full.py se busca desde donde estés situado en la terminal, no desde donde está el script.

Para que SIPA sea profesional e independiente de dónde lo ejecutes, vamos a resolver primero la estandarización de rutas y luego el error de atributo.

* 1. Estandarización de Rutas (Resolviendo el error de Shell)
En tu main_dashboard.py, dentro de run_db_test, vamos a usar la ruta absoluta dinámica. Sustituye la lógica de ese método por esta:

```python
def run_db_test(self):
        """Lanza el test usando rutas absolutas para evitar fallos de entorno."""
        import os
        # Obtenemos la ruta real de la carpeta del proyecto
        base_path = os.path.dirname(os.path.dirname(os.path.abspath(__file__)))
        script_path = os.path.join(base_path, 'test_db_full.py')

        try:
            res = subprocess.run(['python3', script_path], capture_output=True, text=True)
            self.console_audit.config(state=tk.NORMAL)
            self.console_audit.delete(1.0, tk.END)
            self.console_audit.insert(tk.END, res.stdout + res.stderr)
            self.console_audit.config(state=tk.DISABLED)
        except Exception as e:
            tk.messagebox.showerror("Error de Ruta", f"No se encontró el script en: {script_path}")
```

*2. Resolución del Fallo de Atributo (insert_log)
El script test_db_full.py falla porque intenta llamar a db.insert_log, pero en tu versión 0.2.5 de PersistenceManager, ese método probablemente se llama de otra forma (según tu telemetría anterior, usamos métodos como registrar_metrica_arranque).

Para que el test funcione y sea compatible con CITADEL, añade este método a tu clase PersistenceManager en el archivo core/persistence.py:

```python
def insert_log(self, log_data):
        """
        Método de compatibilidad para auditoría externa.
        Inserta un registro manual en la tabla app_logs.
        """
        query = """INSERT INTO app_logs (timestamp, level, message) 
                   VALUES (?, ?, ?)"""
        try:
            cursor = self.conn.cursor()
            # Si log_data es un dict, extraemos, si no, usamos valores por defecto
            ts = log_data.get('timestamp', "N/A")
            lv = log_data.get('level', "INFO")
            ms = log_data.get('message', "Test manual")
            
            cursor.execute(query, (ts, lv, ms))
            self.conn.commit()
            return True
        except Exception as e:
            print(f"Error en persistencia manual: {e}")
            return False
```
